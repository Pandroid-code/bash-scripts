# Bash Scripts Collection

This repository contains a set of Bash scripts I've created for file management, automation, and Linux system tasks. Each script has its own README with more details.

## Scripts Included

1. [rename_vid_series.sh](README_rename_vid_series.md)  
   A script to rename video series files consistently. See the linked README for details and usage instructions.

2. [chk_mode_ext_script.sh](README_chk_mode_ext_script.md)  
   A script to check and adjust Bash script permissions and extensions. See the linked README for details and usage instructions.

## Notes

- All scripts are written in Bash and tested on Linux systems.
- Examples and instructions for running each script are in the individual README files.
